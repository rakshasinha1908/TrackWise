
# 🚀 Trackwise — Smart Expense Analyzer & Tracker

Trackwise is a **modern full-stack expense tracking and financial insights platform** designed to help users **monitor spending, plan budgets, and make smarter financial decisions**.

It goes beyond basic tracking by providing **AI-like insights, predictive analytics, and behavioral patterns** based on user spending.

---

## 📌 Overview

Trackwise allows users to:

* 📊 Track daily expenses with categorized entries
* 📅 Analyze **monthly & weekly spending trends**
* 🎯 Set budgets based on **income & savings goals**
* 📈 Unlock **advanced financial insights**
* 📄 Generate and download **detailed monthly reports**

---

## ✨ Key Features

### 💰 Expense Tracking

* Add, edit, delete expenses
* Categorized spending (Food, Shopping, Bills, Transport, etc.)
* Notes & date tracking for each transaction
* Real-time updates

---

### 📊 Dashboard Insights

* Monthly spending graph
* Weekly spend breakdown
* KPI cards:

  * Total Spend
  * Avg / Day
  * Budget vs Spend
  * Number of Transactions
* Category-wise spending (pie chart)
* Recent expenses overview

---

### 🧠 Smart Insights System

Once the user sets:

* Monthly Income
* Savings Goal
* Category Budgets

👉 Trackwise unlocks **Insights Overview**

#### 🔍 Includes:

* 🧬 **Spending Personality**

  * Example: *“Month-End Splurger”*

* 🌡️ **Category Heatmap**

  * Visual budget utilization per category

* ⚠️ **Budget Overrun Prediction**

  * Predicts overspending before month ends

* ❤️ **Financial Health Score**

  * Composite score (0–100) based on:

    * Budget pacing
    * Savings discipline
    * Category balance
    * Spending consistency

* 📈 **Trend Acceleration**

  * Detects sudden spikes in spending

* 🚩 **Flagged Expenses**

  * Highlights unusual or high-value transactions

---

### 🧾 Monthly Report Generation

Users can download a **complete PDF report** including:

* Total spend, transactions, avg/day
* Budget vs actual comparison
* Category performance
* Financial health score
* Spending highlights
* Full transaction history

## ⚙️ Core Functional Flow

1. User signs up / logs in
2. Adds expenses with categories
3. Sets:

   * Monthly income
   * Savings goal
   * Category budgets
4. Trackwise:

   * Analyzes spending patterns
   * Generates insights
   * Predicts future trends
5. User downloads monthly report

## 📦 Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/rakshasinha1908/TrackWise.git
cd trackwise
```

---

### 2️⃣ Backend Setup

```bash
cd backend
pip install -r requirements.txt
python app.py
```

---

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

### 4️⃣ Environment Variables

Create `.env` file:

```env
SECRET_KEY=your_secret_key
JWT_SECRET=your_jwt_secret
```
